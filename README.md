# 🎮 Premium Minecraft Server Website

This is a modern, responsive, and premium-designed Minecraft server website. It allows players to easily view server IP, commands, member status, store, and leaderboard in a clean and interactive interface.

---

## ✨ Features

- **Premium UI/UX** – Modern and gamer-friendly interface  
- **Live Server Stats** – Shows online status and player count in real time  
- **Commands Guide** – List of all important Minecraft commands  
- **Store Integration** – Dedicated section for server shop/store  
- **Social & Community Links** – Easy access to community platforms  
- **Leaderboard System** – Displays top players in a premium layout  
- **Fully Responsive Design** – Works perfectly on mobile, tablet, and desktop  

---

## 🛠 Technologies Used

- **HTML5** – Structure of the website  
- **CSS3 & Tailwind CSS** – Styling and modern UI design  
- **JavaScript** – Interactivity and dynamic features
