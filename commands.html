Server Commands

Browse all available commands and learn how to use them

/spawn

Essential

Teleport to the server spawn point

Usage: /spawn

/sethome

Essential

Set your home location at your current position

Usage: /sethome [name]
Examples:
  • /sethome - Set default home
  • /sethome base - Set named home "base"

/home

Essential

Teleport to one of your saved home locations

Usage: /home [name]

/tpa

Teleport

Request to teleport to another player

Usage: /tpa <player>
Example: /tpa Steve

/tpaccept

Teleport

Accept a pending teleport request

Usage: /tpaccept

/balance

Economy

Check your current balance

Usage: /balance [player]

/pay

Economy

Send money to another player

Usage: /pay <player> <amount>
Example: /pay Steve 1000

/shop

Economy

Open the server shop GUI

Usage: /shop

/msg

Social

Send a private message to a player

Usage: /msg <player> <message>

/party

Social

Create or manage your party

Usage: /party <create|invite|leave|kick>

/kit

Essential

Claim available kits based on your rank

Usage: /kit [name]

/fly

Admin

Toggle flight mode (VIP+ only)

Usage: /fly

/nickname

Admin

Change your display name (Premium+ only)

Usage: /nickname <name>

/playtime

Essential

Check your total playtime on the server

Usage: /playtime [player]

/vote

Essential

Get voting links and check your vote count

Usage: /vote
